Индивидуальные правки по групповому проекту "Давай поиграем!"

Разработчик: Бадмаева Эльмира
Часть проекта: Игра Крестики-нолики

Исходная работа: Классическая игра "Крестики-нолики" с базовым
функционалом: везде поле 3×3, режим игрок vs компьютер, режим игрок vs
игрок, простая статистика.

Раньше можно было зайти в игру и выбрать в меню варианты: либо игра с
компьютером, либо игра двух игроков, либо просмотр статистики.

Это самая базовая игра в крестики-нолики, и играть в неё очень быстро надоедает (буквально после пары раз), поэтому я подготовила и внесла изменения, которые могут усложнить игру в крестики-нолики и сделать её более интересной.

Внесенные изменения и улучшения:

1. Реализация расширенного игрового поля (Игра Х5)
Добавлен новый режим с полем 5×5 клеток, разработаны новые функции
отрисовки для поля 5×5, реализована проверка победы для 4 символов в ряд,
новый алгоритм для стратегии на большом поле.
Играть на расширенном поле сложнее и интереснее, поэтому игра не так быстро надоедает, а, наоборот, завлекает.

2. Разработка турнирного режима
Добавлена система серийных игр с финальным подсчетом, реализована
система настройки турнира (выбор количества игр), создана отдельная система
учета турнирной статистики, реализована логика досрочного завершения
турнира, разработан финальный экран с результатами.
Добавила турниры для того, чтобы было интереснее играть с компьютером и чтобы было желание выиграть не только одну игру, но и раунд/турнир.

Теперь игра выглядит так: при запуске показывается мини-инструкция, далее в
главном меню варианты режим "игрок vs компьютер", режим "игрок vs игрок",
статистика, вариант обнулить статистику, выход из игры.

Главное изменение: теперь при переходе в режим "игрок vs компьютер" есть
различные варианты игры: "Классика" (обычная игра в поле 3х3), "Игра Х5"
(новая игра в поле 5х5), "Турнир Классика" (турнирный режим в поле 3х3,
можно выбрать количество раундов), "Турнир Х5" (турнирный режим в поле 5х5,
можно выбрать количество раундов).
